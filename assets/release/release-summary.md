# SR8 v1.2.4 Release Summary

SR8 v1.2.4 releases the current local-first OSS compiler surface as:

- product: SR8, an intent compiler for AI systems
- category proof: governed artifact infrastructure through canonical artifacts, validation, lineage, receipts, and package outputs
- bounded horizon: SROS v2 remains the larger governed execution direction and is not claimed as the shipped runtime here

This release includes:

- canonical artifact compilation with deterministic source hashing
- profile overlays, validation, lint, diff, and lineage-safe recompilation
- markdown derivative outputs plus XML package renderers for promptunit_package, sr8_prompt, and safe alternative packages
- chat-frontdoor compile and resume intake flows that terminate in the canonical SR8 artifact path
- breadth registries for entry modes, artifact families, and delivery targets
- local workspace storage with receipts and catalog indexing
- frontend, API, and CLI product surfaces with trusted-local boundaries documented explicitly
- CI, docs, frontend, hygiene, release, and code scanning workflow rails for public OSS release discipline

Use this summary as the release description baseline for the current public repo state.
