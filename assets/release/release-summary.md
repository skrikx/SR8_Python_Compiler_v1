# SR8 v1.0.0 Release Summary

SR8 v1.0.0 delivers a local-first intent compiler with:

- canonical artifact compilation
- profile overlays and validation
- markdown transform targets
- local workspace storage with receipts and catalog
- semantic diff and artifact lint reporting
- CI, release automation, dependency review, and code scanning baseline
- complete OSS documentation and contributor guidance

Use this summary as the baseline release description for the initial public release.
