# Labels

Baseline issue and PR label taxonomy for SR8.

## Type Labels

- `type:bug`
- `type:feature`
- `type:question`
- `type:docs`
- `type:security`

## Priority Labels

- `priority:p0`
- `priority:p1`
- `priority:p2`

## Status Labels

- `status:needs-repro`
- `status:needs-info`
- `status:ready`
- `status:blocked`

## Scope Labels

- `area:cli`
- `area:compiler`
- `area:profiles`
- `area:transforms`
- `area:storage`
- `area:docs`
- `area:ci-release`

Apply labels during triage and keep sets minimal per issue.
