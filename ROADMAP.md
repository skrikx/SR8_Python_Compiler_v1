# Roadmap

This roadmap describes planned direction, not release guarantees.

## Current Baseline (v1.0.0)

- Local-first compile pipeline
- CLI and Python API surface
- Profile overlays and transform targets
- Storage, receipts, diff, and lint
- CI/release/security automation
- Public OSS docs and contributor baseline

## Near-Term Priorities

1. Expand profile quality and validation depth.
2. Improve transform coverage and output quality.
3. Increase fixture breadth for edge-case intent inputs.
4. Strengthen artifact compatibility guarantees.

## Mid-Term Priorities

1. Improve extension ergonomics for profiles and transforms.
2. Add additional deterministic reporting surfaces for auditing.
3. Improve performance profiling for large intent payloads.

## How Roadmap Changes Are Accepted

Roadmap changes are discussed in issues or pull requests and governed by [GOVERNANCE.md](GOVERNANCE.md).
